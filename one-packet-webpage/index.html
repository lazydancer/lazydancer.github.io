One Packet Webpage

I don't have many characters to tell you...

Each packet has roughly 1,400 bytes avaliable to be used. This limit is set by the Maximum Transfer Unit (MTU) as part of IEEE. When a file is larger than a packet, it is broken up into multiple packets and re-combined on your computer

To squeeze more data through, everything in compressed using gzip or deflate and uncompressed by your computer.
